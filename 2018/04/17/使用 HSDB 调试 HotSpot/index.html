启动 jdb

我们在使用 IDE 时，可以很方便地在代码中打上断点进行调试，如果不依赖 IDE，我们可以使用 Oracle JDK 自带的 jdb 工具来完成这项任务。

启动 jdb 时，设定 java 程序使用 Serial GC 和 10MB 的堆内存。

使用 stop in 命令在指定的 java 方法入口处设置断点。

使用 run 命令指定主类来启动 Java 程序。

使用 next、step 等命令来控制前进。
